# Expense-tracker
## 📌 Project Overview

Expense Tracker is a simple Python application that helps users record, manage, and analyze their daily expenses. It allows users to add, view, search, and delete expenses while maintaining a record of their spending.

This project is ideal for beginners learning Python, file handling, and basic data management.

---

## 🚀 Features

- Add new expenses
- View all expenses
- Search expenses by category or date
- Delete expenses
- Calculate total expenses
- Save data using CSV/Text file
- Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Python 3.x
- CSV Module
- Datetime Module
- File Handling

## 💾 Data Storage

The application stores expense records in a CSV file.

Each record contains:

- Date
- Category
- Description
- Amount

Example:

| Date | Category | Description | Amount |
|------|----------|-------------|-------:|
| 2026-07-12 | Food | Lunch | 250 |
| 2026-07-12 | Transport | Bus Ticket | 60 |
